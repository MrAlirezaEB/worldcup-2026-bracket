# BUGLOOS WorldCup 2026 Bracket

این پروژه یک اپلیکیشن پیش‌بینی مرحله گروهی جام جهانی ۲۰۲۶ است که با استفاده از Node.js (بک‌ند) و Vue.js (فرانت‌ند) ساخته شده است.

## ویژگی‌ها
- ورود با ایمیل (هر ایمیل فقط یک بار مجاز به ثبت است)
- قابلیت Drag & Drop برای تعیین رتبه‌بندی تیم‌ها در ۱۲ گروه (A تا L)
- ذخیره‌سازی در فایل JSON (مناسب برای نسخه‌های سبک و تست)
- رابط کاربری مدرن و جذاب با Tailwind CSS v4

## پیش‌نیازها
- Docker و Docker Compose (پیشنهادی برای اجرا)
- Node.js (v18 یا بالاتر - برای اجرای محلی بدون داکر)

## استقرار (Deployment) با Docker

بهترین راه برای اجرای پروژه در محیط تولید استفاده از Docker است.

### استفاده از Makefile (پیشنهادی)

برای راحتی بیشتر، دستورات رایج در یک `Makefile` آماده شده‌اند:

```bash
# بیلد کردن تصاویر (Images)
make build

# اجرا کردن کانتینرها در پس‌زمینه
make up

# مشاهده لاگ‌ها
make logs

# متوقف کردن کانتینرها
make down
```

### استفاده مستقیم از Docker Compose

اگر `make` روی سیستم شما نصب نیست، می‌توانید مستقیماً از دستورات داکر استفاده کنید:

```bash
# بیلد و اجرا
docker-compose up --build -d
```

پس از اجرا، اپلیکیشن در آدرس `http://localhost` در دسترس خواهد بود.

### تنظیم دامین (Domain)
برای استفاده از دامین واقعی در محیط تولید، فایل `docker-compose.yml` را ویرایش کرده و مقدار `DOMAIN` را از `localhost` به دامین خود (مثلاً `example.com`) تغییر دهید. Caddy به صورت خودکار گواهینامه SSL (Let's Encrypt) را برای دامین شما فعال می‌کند.
فراموش نکنید که پورت‌های ۸۰ و ۴۴۳ باید روی سرور باز باشند.

## نحوه اجرا (بدون Docker)

### ۱. تنظیمات
به صورت پیش‌فرض برنامه در پورت 5000 اجرا می‌شود. می‌توانید این مقدار را در فایل `backend/.env` تغییر دهید:
```env
PORT=5000
```
داده‌ها در فایل `backend/db.json` ذخیره می‌شوند.

### ۲. نصب و اجرا
در پوشه اصلی پروژه دستورات زیر را اجرا کنید:

```bash
# رفتن به پوشه بک‌ند
cd backend

# نصب وابستگی‌ها
npm install

# اجرای سرور
node server.js
```

حالا اپلیکیشن در آدرس `http://localhost:5000` در دسترس خواهد بود.

## ساختار پروژه
- `backend/`: شامل کد سرور اکسپرس (Node.js) و ذخیره‌سازی داده‌ها در فایل `db.json`
- `frontend/`: شامل کد Vue 3 و استایل‌های Tailwind. در محیط داکر، از Caddy برای سرو فایل‌ها استفاده می‌شود.
